# 00 - Project Brief

## Proyecto

Dashboard BI + IA para Pollos y Parrillas El Cazador, cadena de restaurantes ubicada en Chilca, Huancayo, Perú.

## Problema de negocio

La empresa opera tres locales y necesita pasar de una gestión empírica a una toma de decisiones basada en datos. El problema se centra en:
- Predicción de ventas diarias y mensuales.
- Optimización de inventario de insumos críticos: pollo, papas y carbón.
- Planificación de personal según picos de demanda.
- Identificación de patrones por local, temporada, descuento, plato y categoría.
- Apoyo a decisiones comerciales mediante dashboards e IA conversacional.

## Locales oficiales

1. Calle Real #232, Chilca
   - Local matriz / consolidado.
   - Aproximadamente 12 años de servicio.
   - Mayor estabilidad histórica.

2. Avenida Huancavelica #587, Chilca
   - Local de crecimiento.
   - Entre 6 y 8 años de servicio.
   - Flujo comercial y vehicular dinámico.

3. Calle Real #976, Chilca
   - Local reciente / expansión.
   - Entre 3 y 5 años de servicio.
   - Mayor variabilidad y necesidad de seguimiento predictivo.

## Nota de normalización

El notebook usa nombres distintos para los locales:
- Ferrocarril & Real
- Av. Huancavelica 436
- Centro

Para la web final se deben usar los nombres oficiales del informe, manteniendo los IDs y factores de venta del notebook cuando sea útil.

## Resultado esperado

Una aplicación web ejecutable localmente con backend y frontend, capaz de generar datos, analizarlos, entrenar modelos y mostrar resultados de forma profesional.
