## Paleta visual

- Marrón oscuro: identidad de pollería.
- Ámbar / amarillo: énfasis.
- Crema: fondo.
- Gris oscuro: texto.
- Verde: métricas positivas.
- Rojo / naranja: alertas.

## Narrativa

El dashboard debe contar esta historia:

1. La empresa tiene tres locales y necesita decisiones basadas en datos.
2. Los datos se centralizan en una arquitectura BI.
3. El análisis muestra patrones de ventas, margen, categorías y locales.
4. K-Means segmenta los platos para tomar decisiones comerciales.
5. La regresión lineal predice ventas mensuales.
6. El simulador convierte el modelo en decisiones de inventario, personal y marketing.
7. El asistente IA facilita consultas rápidas para gerencia.

## Gráficos

Dashboard:
- Línea de ventas anuales.
- Barras de ventas por local.
- Barras horizontales por categoría.
- Tarjetas KPI.
- Gráfico de margen anual.

Exploratorio:
- Histograma de ventas.
- Boxplot o resumen de cuartiles por local.
- Tabla de estadísticas descriptivas.
- Dispersión descuento vs margen.

Segmentación:
- Scatter margen vs frecuencia.
- Tabla de platos por cluster.
- Barras de platos por cluster.
- Tabla de centroides.

Predicción:
- Scatter real vs predicho.
- Residuos.
- Histograma de residuos.
- Barras de coeficientes.

Asistente:
- Chat sencillo.
- Sugerencias de preguntas.
- Respuestas con datos reales.
